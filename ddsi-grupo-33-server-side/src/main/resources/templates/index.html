¿Por qué MetaMapa?

La inteligencia colectiva permite resolver problemas a partir del aporte de información individual. MetaMapa se encarga de ordenar, etiquetar y ubicar esta información para que sea confiable y útil.

Esto permite, por ejemplo, reportar focos de incendios, visibilizar causas, identificar focos de contaminación o proporcionar datos valiosos para resolver desapariciones.
